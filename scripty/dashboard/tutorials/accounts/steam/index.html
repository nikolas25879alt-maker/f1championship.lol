Logging into Steam with Tokens

How to use captured Steam refresh tokens to access accounts

Prerequisites

  • Python 3.6+
  • Steam installed on your machine

How It Works

To log into a Steam account with a refresh token, the script:

  1. Kills any running Steam processes
  2. Extracts the Steam ID from the JWT token
  3. Modifies the Windows registry to set the auto-login user
  4. Creates the required VDF config files with the encrypted token
  5. Launches Steam, which automatically logs in

Copy or download the script below and run it. Enter the username and token from the hit.
